# Linear Equation Solver of Three Variable:
  It helps us to solve linear equations of three variables.
  
# Tech Stack:
  Python
 
# Requirements:
  - Python above 3.7 version
       - To install matplotlib in python we use:
       -  ```pip install matplotlib```
       -  ```pip install numpy```
