# Space Bullet shooter game:
  It is a space shooter game made by using pygame. 
  Move around by using Arrow keys
  Shoot using spacebar.
  
# Tech Stack:
  Python
 
# Requirements:
  - Python above 3.7 version
       - To install pygame in python we use:
       -  ```pip install pygame```
