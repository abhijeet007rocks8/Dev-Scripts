# SPACEWAR Game
## Tools Used:
  * ### pygame
  * ### python 3.9
  * ### pycharm 2021.2.3

## Before Run:
  * install python 3.6 or above.
  * install pygame package using .
    * pip install pygame

## Controls:
 ### Guardian:
    * ⬅️ - left
    * ➡️ - right
    * ⬆️ - up
    * ⬇️ - down
    * fire - RCTRL
 
  ### Aliens:
    * A - left
    * D - right
    * W - up
    * S - down
    * fire - Space Key
