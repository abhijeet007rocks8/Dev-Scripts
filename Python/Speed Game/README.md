# Speed Game:
  It is a Game Where we have to identify a country's flag within a specific time limit.
  
# Tech Stack:
  Python
 
# Requirements:
  - Python above 3.7 version
       - To install Pandas in python we use:
       -  ```pip install Pandas```

Clone this repository:

```
 cd Dev-Scripts
 cd Python
 cd Speed Game
```
