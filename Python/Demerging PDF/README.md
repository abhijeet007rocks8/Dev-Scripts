# Demerging PDF:
  It helps us to demerge a pdf without affecting the main pdf.
  
# Tech Stack:
  Python
 
# Requirements:
  - Python above 3.7 version
       - To install PYPDF2 in python we use:
       -  ```pip install PYPDF2```
