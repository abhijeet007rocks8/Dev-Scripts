# PNG to Icon:
  It is a Python Script to convert a PNG image to a Icon Format.
  
# Tech Stack:
  Python
 
# Requirements:
  - Python above 3.7 version
       - To install Tkinter in python we use:
       -  ```pip install Tkinter```
