# Quick start (how to run locally):

Clone this repository:

git clone ```https://github.com/abhijeet007rocks8/Dev-Scripts.git```
- Change directory:
```
 cd Dev-Scripts
 cd Python
 cd Match Maker
```
- Find main.py and run it.
- ``` python main.py ```
