## QR Code Generator

This is a simple QR code generator. User can scan this QR codes using phone.

### Tech Stack:
<ul>
    <li>Html5</li>
    <li>Css</li>
    <li>Javascript</li>
</ul>

## **Get started**
- Clone this repository

``` 
git clone https://github.com/abhijeet007rocks8/Dev-Scripts.git
```
- Change Directory

```
cd Web Development
```
```
cd QRcode_Generator
```
```
cd index.html
```
