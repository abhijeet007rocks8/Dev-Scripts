# Graph

## Tech Stack
HTML CSS JS PHP

### Description
Draw the Graph Of any Arithmatic Function.

### Try : 
``x = y + 100``   
``x = cos(x) + 10``

``cd Random Anime profile``   

### Instruction :
Move the Folder Graph to the htdocs folder inside XAMPP.
