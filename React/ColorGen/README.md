## Description 
You can create variety of shades of one hex value. More clearly, you will not get only one color, but you will get all the shades of the color whose hex value you will submit to our website.

## Tech Stack
1. ReactJS
2. ValueJS
3. CSS

## Basic Components

Just Use `npm install` after reaching the directory folder in terminal to install all the package dependencies

## Quick Start (How to run in Local System)

1. Clone the Repo using `git clone https://github.com/abhijeet007rocks8/Dev-Scripts.git`
2. Then cd `ReactJS`
3. cd `ColorGen`
4. Then in the vscode terminal use `npm i` to install all the package dependencies
5. Then Use `npm start`
6. Enjoy
